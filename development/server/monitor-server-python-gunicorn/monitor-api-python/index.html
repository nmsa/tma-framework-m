TMA-Monitor - ATMOSPHERE

The sources of TMA Framework components are available at https://github.com/eubr-atmosphere/tma-framework/.
Find more about the project at the website http://atmosphere-eubrazil.eu.

If you are reading this, you are probably looking to use TMA-Monitor thorough REST.
For this, you should use the HTTP method post to the url: http://ip:port/monitor/.
Our libraries will help you with that. There libraries for the languages Java, C# and Python.

If you use Java, the library is presented in this link. The instructions of how to use it and how to include it in your project is presented in this README file

If you are using Python programming language, the library can be found here. All the instructions needed to use this library can be found in this README file.

If your project is being developed in C# you can use the C# library to communicate correctly with this Monitor. All instructions needed can be found in this README file

You can use this form to test the REST endpoint.
